Commandes bots sur le serveur

		

				

    
Nintendo 3DS et DSi

    

		
DS-Homebrew Wiki

	
    
    
  • !forwarders : affiche le guide sur l'utilisation des forwarders.
    • 
    
  • !twlskins : affiche la base de données de skins (sous-thèmes) du TWiLightMenu++.
    • 
    
  • !boxartguide : affiche le guide sur l'ajout de boxarts dans le TWiLightMenu++.
    • 
    
  • !widescreen : affiche le guide sur l'utilisation du mode plein écran avec nds-bootstrap (uniquement sur 3DS)
    • 
    

    

    
Aide

   
    
     
  • !nonbrick : affiche le guide lorsque la LED bleue de votre console s'allume puis s'éteint immédiatement.
    • 
     
  • !sdlock : affiche le guide en cas de problème de lecture de Carte SD.
    • 
     
  • !lumaconfig : affiche le guide de configuration de Luma.
    • 

  • !root : qu'est-ce que la racine d'une carte SD.
    • 
	   
  • !updateluma : affiche un tutoriel sur comment mettre à jour Luma3DS.
    • 
		

	   
Recherche d'applications

	   
    
	   
  • .udb [nom de l'app] : recherche une application dans la base de données Universal-DB (Universal-Updater).
  • 
	   
  • .gamebrew [nom de l'app] : recherche une application dans la base de données Gamebrew.
    • 
		

				

				

	   
Nintendo Switch

	   
    
		   
  • /imsp |NUMERO DE SERIE SWITCH| : vérifie si votre Switch est compatible avec la faille RCM.
    • 
		   
  • !nxcheck : affiche où se situe la numéro de série de votre Nintendo Switch.
    • 
		   
  • !ftpdswitchandroid : affiche comment configurer FTPD sur votre Nintendo Switch avec votre appareil Android.
    • 
		   
  • !ftpdswitchwindows : affiche comment configurer FTPD sur votre Nintendo Switch avec votre ordinateur sous Windows.
    •
